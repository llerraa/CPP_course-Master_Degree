Для сдачи лабораторных работ по C++ в магистратуре РУДН

**Лабораторная работа №1: Трогаем C++**
Темы: расзработать простейший калькулятор

**Лабораторная работа №2: Основы C++**
Темы: Типы данных, массивы, пользовательские типы данных, сложные типы данных, ссылки и указатели.

**Лабораторная работа №3: Объектно-Ориентированное Программирование в C++**
Освоить основные принципы ООП в C++: инкапсуляцию, наследование, полиморфизм и абстракцию на практике.

**Лабораторная работа 4: банковская система (в отдельном репозитории)**
Создание объектно-ориентированного приложения, демонстрирующего практические навыки работы со структурами данных, классами и организацией проекта на C++.
Технологический стек: C++ (стандарт C++17 или выше), CMake, кроссплатформенность (Windows/Linux/macOS)

**Лабораторная работа №5: Многопоточность и асинхронное программирование в C++**
Закрепить практические навыки работы с потоками, синхронизацией и асинхронным программированием в C++.

**Лабораторная работа №6: Работа с базами данных в C++**
Цель работы: Освоить практические навыки работы с базами данных в C++ using SQLite, изучить основные операции CRUD, транзакции, безопасность и оптимизацию запросов.
